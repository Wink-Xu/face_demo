# Face Recognition Demo

A small demo to compare the the similarity of two image .

## Content

├── faceRecognition
│   ├── core
│       ├──  getKeypoints.py
│       ├──  alignFace.py
│       ├──  getFeature.py
│       ├──  faceRecognition.py

│   ├── results
│   ├── sample


## Usage

Three process 1.keypoints 2.align 3.feature
```
cd core
sh run.sh
```
Compare the similarity of two image
```
cd core
sh compare.sh
```
